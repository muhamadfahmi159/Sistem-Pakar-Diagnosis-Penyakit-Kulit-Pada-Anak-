# Sistem-Pakar-Diagnosis-Penyakit-Kulit-Pada-Anak
There are 3 types of skin diseases in children caused by viruses, namely:
1. Measles
2. German measles (Rubella)
3. Chickenpox
Expert system will help medical practitioner diagnosing those diseases.

The NER model is able to detect the symptoms entities with ALBERT model reached the highest F1 Score of 0.83.
This experiment succeeded in creating a web-based system that is easier for users to use.
The implementation of the system is able to produce a diagnosis of children skin diseases based on symptom input from free text.
